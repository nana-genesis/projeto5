# Lista de Contatos

Projeto React para cadastro de contatos, desenvolvido como exercício com Redux e Styled Components.

## Funcionalidades

- Adicionar contato com nome completo, e-mail e telefone
- Editar contatos cadastrados
- Remover contatos da lista
- Gerenciar os dados com Redux Toolkit
- Estilizar a interface com Styled Components

## Tecnologias

- React
- Redux Toolkit
- React Redux
- Styled Components
- Vite

## Como executar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
